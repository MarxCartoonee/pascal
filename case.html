Наредба вишеструког гранања

Шта ће се исписати извршавањем следећих наредби:
write('Redni broj: '); slovo:='E';
case slovo of
'A':writeln('1');
'E':writeln('5');
'I':writeln('9');
'O':writeln('15');
'U':writeln('U');
end;


Колике ће бити вредности променљивих после извршавања следећих наредби
p:=true; d:=1;
case k mod 10 of
3,2,7,5:d:=k;
1:; 4,8:begin p:=false; d:=2 end;
9,6:begin p:=false; d:=3 end;
end;
ако целобројна променљива k има следеће вредности:
а)6
б)235
в)71
г)100


Наредбу условног преласка којом се израчунава максимум два броја
if a>b
then max:=a
else max:=b
заменити еквивалентном наредбом избора case.


Написати case наредбу еквивалентну следећој if наредби:
if k<0
then r:=r+1
else if k=1
then s:=s+1
else if (k=2) or (k=3) or (k=4)
then t:=t+2


Написати наредбу case којом се на основну бројних оцена 5, 4, 3, 2, 1 исписују описне оцене 'одличан', 'врло добар', 'добар', 'довољан', 'недовољан'.
