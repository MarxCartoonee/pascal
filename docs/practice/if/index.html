Условна наредба

Које ће вредности имати променљиве j и k после извршења наредбе условног преласка:

if j>k
then j:=k
else k:=k-1

ако су вредности променљивих:

  1. j=2, k=4
  2. j=2, k=2
  3. j=4, k=2

Написати наредбу условног преласка којом се вредност променљиве f израчунава по формули:

f=a+b, ако је а непарно;
f=a*b, ако је а парно

За које вредности променљиве а нису еквивалентне наредбе условног преласка:

  1. if a>0 then b:=2 else b:=5
  2. if a<0 then b:=5 else b:=2

Коју ће вредност имати променљива x по извршењу следећих наредби:

  1. x:=2;
    if x<3 then x:=x+3;
  2. x:=2; y:=1;
    if x+y<=x-y
    then x:=5
    else x:=+5;
  3. x:=1; y:=-5
    if abs(y)>2
    then
    if y<0
    then x:=0
    else x:=2;
  4. x:=10; y:=2;
    if abs(y)>3
    then begin
    if y<0
    then x:=3
    else x:=4
    end;
  5. x:=10; y:=2;
    if abs(y)>3
    then
    begin if y<0 then x:=3 end
    else x:=4;

Да ли се следећом наредбом условног преласка:

if a<=6
then c:=1
else if a>6
then c:=2
else c:=3

може променљивој a доделити вредност 3? Како се може упростити дата наредба?
