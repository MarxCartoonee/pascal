Условна наредба


Условна наредба рачуна Булов израз и, у зависности од резултата, извршава или прескаче наредни корак.


IF 'услов' THEN 'следећи корак';
IF x<2 THEN WRITELN (x);


Према примеру, ако је x мање од 2 програм ће га исписати и продужити са следећим кораком, у супротном ће одмах прећи на следећи корак.


Графички симболи условне наредбе су



Који ће се користити зависи само од неопходног простора.


Условна наредба такође има и продужени облик који омогућује алтернативну путању уколико је вредност Буловог израза ⊥.


IF 'услов' THEN 'следећи корак' ELSE 'алтернативни корак';
IF x=5+2 THEN WRITELN (x) ELSE WRITELN (5+2);


Сигурно сте приметили да се ; ставља тек на крају последњег корака. Од овога се одступа једино ако се под условом појаве две или више програмских линија. Тада се те програмске линије морају повезати са BEGIN и END;. Ово се такође примењује код CASE, FOR и WHILE наредби.


IF 'услов' THEN BEGIN 'први корак'; 'други корак'; END;
IF 'услов' THEN 'следећи корак' ELSE BEGIN 'први алт. корак'; 'други алт. корак'; END;
IF x>12 THEN BEGIN y:=x; WRITELN (x+1); END;
IF y<=a THEN WRITELN (a) ELSE BEGIN WRITELN (y); WRITELN (y-a); END;


Сада ћете видети зашто је ово битно.


IF y<=a THEN WRITELN (a) ELSE WRITELN (y); WRITELN (y-a);


У овом случају би програм исписао y-a независно од вредности Буловог израза.
